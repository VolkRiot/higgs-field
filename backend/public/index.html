Higgs Field API

  • /microservices
    • GET

      Returns all Microservices

      • /add
        • POST

          Adds a single Microservice

      • /get/:id
        • GET

          Returns a single Microservice by ID

      • /delete/:id
        • GET

          Deletes a single Microservice by ID

      • /microservices/get/:id/endpoints
        • GET

          Returns all End Points by Microservice ID

  • /dbconnections
    • GET

      Returns all DB Connections

      • /get/:id
        • GET

          Returns a single DB Connection by ID

  • /endpoints
    • GET

      Returns all End Points

      • /add
        • POST

          Adds a single End Point

      • /get/:id
        • GET

          Returns a single End Point by ID
